Com base no livro do Robert C. Martin (“Uncle Bob”)  o autor revela soluções simples e diretas para os desafios reais que você enfrentará 
e que irão influenciar diretamente o sucesso ou fracasso dos seus projetos.

Este projeto foi desenvolvido em NodeJs com TypeScript seguindo muitos conceitos apresentados pelo autor entre alguns deles temos:

- Conceitos do SOLID
- Uso de abstrações ao invés de classe concreta
- Desacoplamento

O projeto conta ainda com Test unitário para que possamos ter segurança na hora de alterar o código.

<br/>

## 📋 Instalação

⚠️  
- É necessário ter no [NojeJs](https://nodejs.org/pt-br/) e o [Yarn](https://classic.yarnpkg.com/lang/en/) 
- Ambos devem instalados em sua máquina
- Feito isso abra o terminal e execute o seguinte comando:

```
yarn
```

<br/>


## 😎 Iniciando o projeto

- Agora para rodar o projeto local execute o comando:

```
yarn dev
```

- E para rodar o test unitário execute o comando:
```
yarn test
```
